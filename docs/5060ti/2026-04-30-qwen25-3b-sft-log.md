# 2026-04-30 Qwen2.5-3B LoRA SFT 实验记录

这份记录对应一次在当前单卡环境上完成的 `Qwen2.5-3B-Instruct + LoRA` `SFT` 实验。

这轮实验的目的不是直接进入偏好优化，而是回答一个更现实的问题：

- `3B` 级别模型是否已经能稳定承载当前任务
- 相比 `0.5B`，是否更值得作为下一阶段主实验模型

## 2. Smoke 结果

smoke 训练最终指标：

- `epoch = 20.0`
- `train_loss = 1.7067`
- `eval_loss = 1.6775`
- `train_runtime ≈ 5m48s`

这说明：

- `3B` 在当前模板、数据、LoRA 配置下能稳定启动并完成训练
- smoke 没有出现 OOM、tokenizer、模板或 checkpoint 相关错误

## 3. Full 结果

full SFT 最终指标：

- `train_loss = 1.9460`
- `eval_loss = 1.6482`
- `train_runtime ≈ 9m53s`
- `eval_samples = 48`

从输出目录可确认：

- adapter 已正常保存
- `training_loss.png` / `training_eval_loss.png` 已生成
- 后续已经具备进入推理评测阶段的条件

## 4. 结果解读

最直接的观察是：

- `3B smoke` 和 `3B full` 都稳定跑通
- `3B full eval_loss = 1.6482`

相对于此前 `0.5B full` 的 `eval_loss ≈ 1.935`，这是一个明显更强的收敛信号。

需要克制的是：

- loss 更低不等于内容质量一定更好
- 真正是否值得继续投入，还需要经过：
  - 自动规则评测
  - pairwise judge

## 5. 当前阶段结论

这轮 `3B` 训练已经足够回答一个关键问题：

**`Qwen2.5-3B-Instruct + LoRA` 可以稳定承载当前任务，而且比 `0.5B` 更值得作为下一阶段主实验候选。**

## 7. 自动规则评测结果

基于 `48` 条 eval 集，当前 `3B` 默认 prompt 路线已经完成了第一轮自动规则评测。

结果摘要如下：

### `qwen25_3b_default_prompt_base`

- `json_parse_success = 43 / 48 = 89.58%`
- `schema_valid = 0 / 48`

这说明：

- `3B` base 已经具备较强的“按 JSON 形式回答”的能力
- 但由于默认 prompt 没有强约束固定字段协议，base 仍不会稳定命中目标 schema

### `qwen25_3b_default_prompt_lora`

- `json_parse_success = 47 / 48 = 97.92%`
- `schema_valid = 47 / 48 = 97.92%`
- `category_matches_reference = 35 / 47 = 74.47%`
- `severity_matches_reference = 31 / 47 = 65.96%`

这说明：

- `3B + LoRA` 基本已经把“合法 JSON”推进到“严格命中目标 schema”
- 相比 `0.5B`，`3B` 的基础能力和微调上限都更强

## 8. 与 `0.5B` 的阶段性对照

当前至少可以得出一个非常清楚的观察：

- `0.5B default_prompt_base` 的 `json_parse_success` 几乎为 `0`
- `3B default_prompt_base` 的 `json_parse_success` 已接近 `90%`

这说明：

- `3B` base 和 `0.5B` base 已经不是一个量级
- `3B` 更适合作为下一阶段真正的主实验候选

同时：

- `3B default_prompt_lora` 的 `schema_valid` 已接近 `98%`

这表明 `3B` 不仅更会“输出 JSON”，也更容易被 LoRA 拉到目标任务协议上。

## 9. 当前建议

下一步最值得做的事情是：

1. 用 pairwise judge 比较 `0.5B default_prompt_lora` vs `3B default_prompt_lora`
2. 判断 `3B` 的内容质量是否也显著优于 `0.5B`
3. 如果 judge 结果同样明显占优，就把 `3B` 确认为后续偏好优化主线模型
